Projects

  • Bomb-Disposal-Robot

    Bomb Disposal robot

    The Bomb Disposal Robot project is an innovative solution designed to handle hazardous objects remotely, ensuring safety and efficiency. Utilizing an Arduino board and a NodeMCU, this robot can be controlled via the Blynk cloud, allowing for precise operations through a mobile app. The robot features three independently controlled robotic fists and a mobile base that can move in all directions. Each fist is capable of rotating, gripping, releasing, and moving up and down, providing versatile manipulation capabilities.

    This project integrates several components, including an Arduino board, NodeMCU, and various sensors and motors, all of which work together seamlessly. The NodeMCU reads commands from the Blynk app and communicates them to the Arduino, which executes the actions. The setup process involves configuring the hardware, setting up the Blynk account, and uploading the necessary sketches to the Arduino and NodeMCU. This project not only demonstrates the effective use of IoT and cloud technology but also highlights the potential for remote control applications in safety-critical environments.

  • Wind Farm Power Forecasting

    Wind Power Forecasting

    The Wind Farm Power Forecasting project is a web API designed to predict wind farm power production using XGBoost, a machine learning algorithm. Accurate forecasts are essential for optimizing power systems and integrating renewable energy into the grid. The project utilizes data from the Chinese State Grid Renewable Energy Generation Forecasting Competition, focusing on three wind farms. The system involves extensive data analysis, model training, and a user-friendly web interface developed with Flask, allowing users to input wind farm IDs and forecasting horizons for power predictions. The project also includes comprehensive documentation and a Postman environment for easy API testing.



  • Iot Agriculture system

    IOT Agriculture System

    The IoT-Based Agriculture Monitoring System project aims to provide farmers with real-time monitoring of their fields remotely. By utilizing sensors to measure soil moisture, humidity, and water levels, this system transmits data to a cloud platform for easy access and visualization. The project integrates a NodeMCU, Arduino, and various sensors to ensure comprehensive field monitoring. Additionally, the system features computer vision-based soil moisture detection using OpenCV on a Raspberry Pi, enhancing soil moisture analysis and providing critical alerts through a user-friendly dashboard. This project exemplifies the integration of IoT and cloud technologies for efficient agricultural management.
